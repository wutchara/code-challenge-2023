# Reverse Link List

Given the head of a singly linked list, reverse the list, and return the reversed list.


Example 1:

![img-2](./rev1ex1.jpg)

Input: head = [1,2,3,4,5]

Output: [5,4,3,2,1]

<br />
<br />

Example 2:

![img-2](./rev1ex2.jpg)

Input: head = [1,2]

Output: [2,1]
